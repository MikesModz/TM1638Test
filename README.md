# TM1638 Test

Test project for driving TM1638 seven segment display driver with key scan input. Implemented on a ATMega328 
micro controller (Arduino Nano clone). 

[TM1638 LED and KEY Display](http://www.ebay.co.uk/itm/181874730941?_trksid=p2057872.m2749.l2649&ssPageName=STRK%3AMEBIDX%3AIT)

##Arduino Nano Connections

Connections     | Arduino Pin
:----------------|:-----------
| CLOCK         | A0 (PC0) |
| STROBE        | A1 (PC1) |
| DATA          | A2 (PC2) |
